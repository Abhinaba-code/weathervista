# **App Name**: WeatherVista

## Core Features:

- City Weather Search: Allow users to search for weather information by city name using a search bar. Uses OpenWeatherMap API.
- Current Weather Display: Display current weather conditions including temperature, feels like, humidity, wind speed, weather icon/description, UV index, visibility, and pressure.
- 5-Day Weather Forecast: Show a 5-day weather forecast, including daily high and low temperatures and weather icons.
- 24-Hour Weather Forecast: Provide an hourly weather forecast for the next 24 hours, displayed as individual cards.
- Dynamic Background: Implement a gradient background that dynamically changes based on current weather conditions (sunny, rainy, cloudy, snowy).
- Location & Time Display: Show the current location details and the current date/time.
- Weather Alerts Display: Implement a tool to process weather alerts if there are any applicable alerts. These will be based on location.
- Recent Searches: Show the most recent successful searches at the bottom of the page. Clicking on a city's name in the "Recent Searches" section will automatically conduct a new search.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6), inspired by a clear sky, to promote feelings of openness and tranquility.
- Background color: Light, desaturated blue (#E3F2FD) to maintain a light and airy feel, complementing the primary color.
- Accent color: Muted purple (#9575CD) to give a distinctive, refined appearance and highlight important data and interactive elements.
- Headline font: 'Space Grotesk', a geometric sans-serif, is for headlines and emphasized data displays, like current temperature readings.
- Body font: 'Inter', a grotesque sans-serif, is for displaying the other information on the screen. It goes well with Space Grotesk.
- Use a set of modern, animated weather icons and illustrations. Select iconography to work in light and dark mode.
- Apply a glassmorphism effect to the forecast cards and panels to create a modern, frosted-glass appearance.
- Add smooth transitions and animations when the weather data updates or when the user interacts with the interface.
